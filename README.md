# Rostering Landing Page

Modern healthcare rostering and workforce management landing page.

## Live Demo

https://enzosureta.github.io/rostering-landing/

## Features

- Responsive landing page
- Healthcare-focused UI
- Staff rostering dashboard
- Analytics section
- Conflict management page
- Leave request page
- Smooth animations
- Mobile friendly

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/enzosureta/rostering-landing.git
```

Open `index.html` in your browser.
